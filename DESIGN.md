# Design System — Minseo9 (6900번 도착 알림)

## Product Context
- **What this is:** 특정 버스 1개의 도착을 지켜보는 개인/가족용 안드로이드 유틸리티 앱. 정류장/버스를 지정하면 모니터링 시작 시 도착까지 남은 시간을 알려준다.
- **Who it's for:** 나 자신과 가족. MinseoStore(개인용 앱 스토어)를 통해 배포.
- **Space/industry:** 대중교통 알림 (카카오버스, 네이버지도 대중교통과 인접 카테고리).
- **Project type:** 단일 화면 네이티브 안드로이드 앱 (웹 아님).
- **핵심 인상 (memorable thing):** "버스 놓칠 걱정 없이 안심하게 기다릴 수 있다" — 모든 디자인 결정은 이 문장을 기준으로 판단한다.

## Aesthetic Direction
- **Direction:** Calm Utilitarian (차분한 유틸리티)
- **Decoration level:** minimal — 카운트다운 숫자와 타이포그래피가 대부분의 일을 한다. 아이콘 원, 그라디언트, 장식용 도형 없음.
- **Mood:** 여러 노선을 한 번에 보여주는 밀도 높은 대중교통 앱(카카오버스/네이버지도)과 달리, 이 앱은 버스 1개만 본다. 따라서 정보 밀도가 아니라 여유와 안정감이 핵심.
- **왜 카테고리 관행과 다른가 (EUREKA):** 대중교통 앱들은 여러 노선을 탐색하는 용도라 밀도 높은 리스트형 UI가 맞다. 이 앱은 사용자가 이미 무엇을 기다리는지 알고 있으므로, 밀도 대신 "하나의 큰 숫자 + 여백"이 맞는 축이다. 색상도 급함(빨강/주황 알람색)이 아니라 안정감을 주는 파랑/틸 계열로 간다 — 목표가 "안심"이지 "긴급 알람"이 아니기 때문.
- **참고:** 카카오버스(레드닷 수상, 밀도 높은 리스트, 승하차 알림), 네이버지도 대중교통(디테일 있는 UX). 이 앱은 참고하되 밀도/색상 방향은 의도적으로 다르게 간다.

## Typography
- **Display/Hero (카운트다운 숫자):** Pretendard, ExtraBold(800), ~56–96sp — 화면에서 가장 중요한 단일 요소.
- **Body (부가 정보 — 정류장 수, 잔여좌석):** Pretendard, Medium(500), 15–17sp.
- **Label (조용한 라벨 — 노선/정류장명):** Pretendard, Bold(700), 13sp, 약간의 자간.
- **Android 적용:** Pretendard 폰트를 `res/font/`에 variable/static 폰트 리소스로 번들 (Downloadable Fonts API 대신 직접 번들 — 개인용 앱이라 네트워크 의존 없이 확실하게).
- **금지:** 기본 시스템 폰트(Roboto 기본값) 그대로 사용 금지. Noto Sans KR 기본 느낌도 지양.

## Color
- **Approach:** restrained — 액센트 1개 + 중립 그레이. 색이 드물고 의미 있게 쓰인다.
- **Accent (Calm Blue):** `#1C6E8C` — 기본 상태, 버튼, 카운트다운 숫자.
- **Accent Tint:** `#63A6BF` — 보조 강조.
- **Accent Soft:** `#E4F0F4` — 세그먼트 토글 배경 등 옅은 배경.
- **Warn (임박, 3분 이하):** `#C08A3E` — 의도적으로 빨강/주황이 아닌 차분한 앰버. 급하다는 신호는 주되 "경보"처럼 느껴지지 않아야 한다.
- **Warn Soft:** `#F5EAD9`
- **Text:** `#1A202C` / **Text Muted:** `#5B6B79`
- **Background:** `#F7FAFC` / **Surface:** `#FFFFFF`
- **Dark mode:** Accent와 Accent Tint를 서로 스왑(다크에서 밝은 톤이 액센트), 배경 `#0F1720`, 서피스 `#172230`, 텍스트 `#EDF2F7` / `#94A3B8`, 보더 `#263241`. Android `values-night/`로 구현.

## Spacing
- **Base unit:** 8px
- **Density:** spacious (단일 화면, 밀도보다 여유가 우선)
- **Scale:** xs(4) sm(8) md(16) lg(24) xl(32) 2xl(48) 3xl(64)

## Layout
- **Approach:** 세로 단일 포커스 스택 (리스트/그리드 아님)
- **화면 구조 (위 → 아래):**
  1. 조용한 라벨: 노선명 (`6900 잠실종합운동장 방향`)
  2. 정류장명 (`동천동현대홈타운2차아파트`)
  3. Hero 영역 (화면 중앙, flex-grow) — 대기중일 땐 안내 문구, 모니터링 중일 땐 큰 카운트다운 숫자 + 부가정보
  4. 세그먼트 토글 (이번 차량 / 다음 차량) — 기존 RadioGroup 대체
  5. 하단 전체너비 CTA 버튼 (시작: primary 채움 / 종료: outline)
- **Border radius:** sm(10) md(16, 버튼/토글) lg(20, 카드형 요소가 생긴다면)

## Motion
- **Approach:** intentional — 최소한이지만 의미 있게
- **모니터링 중 표시:** 노선 라벨 옆 작은 펄스 점(pulse dot)으로 "살아있음"을 조용히 표현 (1.6s ease-in-out 반복)
- **숫자 갱신:** 급격한 전환 대신 부드러운 크로스페이드 (구현 시 `TextView` 알파 트랜지션 정도로 충분)
- **임박 전환(3분 이하):** 색상만 앰버로 전환, 깜빡임/진동 애니메이션 없음 — "경보"가 아니라 "차분한 신호"
